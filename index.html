ChartAdvisor: Efficient Algorithm for Recommendation of Data Visualization Tools.

Chart Advisor is a standalone software that processes an Input RDF Dataset and produces as result a ranked list of recommended visualization tools (charts) such as Pie, Column, Line charts (and others). The core algorithm of the software is intended to be used by Linked Data Projects (LinDa) and similar solutions.
